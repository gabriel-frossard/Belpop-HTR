# Belpop-HTR
This repository contains all the scripts used in the Belpop project, a bidisciplinary study using Computer Science for Historical Demography. 
This project aims to extract all the information gathered in civil records of the city of Belfort, France but also the DMC employee files, a former company located in Belfort.

The Belpop pipeline is as following : 

raw scans --> ROI segmentation --> Handwritten Text Recognition (HTR)
